## 1. Vue d'ensemble

**Nova Solo** est une plateforme tout-en-un pour l'entrepreneur solo en Suisse romande. Elle couvre l'intégralité du cycle de vie d'un projet indépendant :

```
Diagnostic → Stratégie → Offre → CV/Dossier → Pipeline → Marketing → Finances → Gestion
```

Chaque module est connecté aux autres. Exemple : le Diagnostic génère automatiquement une première ébauche de ton BMC, Business Plan, CV et Pricing — pas besoin de ressaisir tes informations.

**Ce que Nova Solo contient :**
- **23 modules** couvrant toutes les dimensions du pilotage solo
- **7 agents IA spécialisés** (stratégie, finance, communication, commercial, juridique, technique, coordination)
- **Contexte suisse romand intégré** : taux AVS, TVA 8.1%, LACI art. 71a–71d, CHF, BCVS
- **Données 100% privées** : chaque compte est isolé, aucune donnée partagée entre utilisateurs

## 2. Premiers pas — Configuration initiale

### Étape 1 — Créer ton compte

Accède à l'application et clique sur **"Se connecter"**. Crée un compte avec ton adresse email. Un email de confirmation est envoyé.

### Étape 2 — Remplir les Réglages

> **⚠️ À faire en premier** — La qualité des réponses IA dépend directement de ces informations.

Accède aux **Réglages** (section COMPTE dans la barre latérale) et remplis :

| Champ | Importance | Description |
|---|---|---|
| **Nom complet** | Obligatoire | Utilisé dans tous les documents générés |
| **Statut** | Obligatoire | laci / reconversion / création / existant |
| **Domaine d'activité** | Obligatoire | Ex : "Coaching en insertion professionnelle" |
| **Localisation** | Recommandé | Ex : "Monthey, Valais" |
| **Nom de marque** | Recommandé | Ex : "SeedJobs" |
| **Bio professionnelle** | Recommandé | Résumé de ton parcours et de ton offre |
| **Tarif horaire** | Recommandé | Base pour les calculs financiers |
| **Charges fixes** | Recommandé | Loyer bureau, abonnements, etc. (CHF/mois) |

**Barre de progression** : La barre dans la barre latérale mesure la complétude de ton profil sur 6 étapes clés. Plus elle est remplie, plus les agents IA sont précis.

### Étape 3 — Lancer le Diagnostic

Une fois le profil rempli, va dans **Diagnostic** et réponds aux 3 questions. L'IA génère un plan d'action et pré-remplit automatiquement 4 modules (BMC, Business Plan, CV, Pricing).

## 4. PILOTAGE

### 4.1 Tableau de bord

**Route** : `/` (page d'accueil)

Le tableau de bord est ton cockpit quotidien. Il centralise :

**KPIs financiers** (en haut)
- Chiffre d'affaires du mois
- Dépenses du mois
- Trésorerie disponible

**Rituels quotidiens**
4 actions de momentum à cocher chaque jour :
1. Contacter 1 personne de ton réseau
2. 10 min de visibilité (LinkedIn, post, commentaire)
3. Avancer sur un livrable clé (BMC, BP, offre)
4. Relancer 1 prospect en attente

> Les rituels se réinitialisent chaque jour. Ils ne sont pas stockés en base de données — ils servent de check-list mentale quotidienne.

**Jalons LACI** (si statut = "laci")
6 étapes du dispositif art. 71a–71d pour les demandeurs d'emploi créateurs :
1. Annoncer le projet à l'ORP
2. Obtenir l'accord de l'ORP pour la phase de planification
3. Élaborer le business plan
4. Faire expertiser le projet
5. Déposer la demande de soutien LACI
6. Lancer l'activité avant l'échéance de l'allocation

**Simulateur ROI — Valeur Nova Solo**

Un outil interactif pour mesurer concrètement ce que Nova Solo te rapporte en temps et en CHF.

*Principe* : chaque module remplace une tâche manuelle. Le simulateur calcule les heures récupérées par mois et les traduit en valeur monétaire selon ton taux horaire.

- **Ton taux horaire** : slider de 60 à 350 CHF/h (60 = assistant, 200 = coach, 350 = expert)
  - Valeur par défaut : 120 CHF/h (taux typique d'un coach indépendant romand)
  - Réglage persistant entre sessions
- **Modules de lancement** *(gains étalés sur 3 mois)* : Diagnostic, BMC, Business Plan, Offre & Pricing, CV & Dossier, Vision symbolique
- **Modules quotidiens** *(gains récurrents chaque mois)* : Pipeline, Finances, Comptabilité, Factures, Cabinet Hermès, Marketing

Coche les modules que tu utilises → les résultats s'affichent en temps réel :
- Nombre de modules actifs
- Heures récupérées / mois
- Valeur générée / mois (CHF)
- ROI annuel estimé (CHF)

> **Exemple** : Laure, coach à 120 CHF/h, active Factures (1,5h), Pipeline (3h) et Business Plan (4h sur 3 mois). Nova lui économise **5,8h/mois = 696 CHF/mois** de temps valorisé.

---

**Signaux de bien-être**
4 alertes solo (l'isolement est le risque n°1 de l'entrepreneur solo) :
- Travailler 50h+/semaine depuis 2 semaines
- Aucun contact avec un pair cette semaine
- Incapacité à décrocher le soir/week-end
- Négliger son hygiène de vie (sport, repas, sommeil)

**Ressources locales** : Masterminds, Genilem, CCSO, Innosuisse, espaces de coworking en Suisse romande.

### 4.2 Diagnostic

**Route** : `/diagnostic`

Le Diagnostic est le point de départ recommandé pour tout nouvel utilisateur.

**Comment ça marche :**
1. Réponds à 3 questions (situation actuelle, blocages, vision à 12 mois)

> **Note** : La cascade ne détruit pas le contenu existant. Si tu as déjà rempli un bloc BMC, il n'est pas écrasé.

## 5. STRATÉGIE

### 5.1 BMC — Business Model Canvas

**Route** : `/bmc`

Le BMC structure ton modèle économique en 9 blocs visuels :

| Bloc | Description |
|---|---|
| **Partenaires clés** | Qui t'aide à fonctionner (fournisseurs, alliances) |
| **Activités clés** | Ce que tu fais concrètement pour délivrer la valeur |
| **Ressources clés** | Ce dont tu as besoin (compétences, outils, réseau) |
| **Proposition de valeur** | Ce que tu apportes de unique à tes clients |
| **Relations clients** | Comment tu interagis avec eux (coaching 1:1, groupe, plateforme) |
| **Canaux** | Comment tu touches et livres tes clients |
| **Segments clients** | Qui exactement tu sers (personas, niches) |
| **Structure de coûts** | Tes charges fixes et variables |
| **Sources de revenus** | Comment tu es payé (abonnement, séance, retainer...) |

**Fonctionnalités :**
- Cliquer sur un bloc pour l'éditer directement
- Bouton **"Enrichir avec l'IA"** : l'agent Strategist approfondit le bloc sélectionné selon ton profil
- Bouton **"Mode défi"** : l'IA challenge tes hypothèses et identifie les angles morts
- **Analyse globale** : vision d'ensemble de la cohérence du canvas
- Chaque bloc affiche sa priorité (Haute / Moyenne / Basse) et un guide d'enrichissement

**Route** : `/pricing`

Calculateur de tarification et de rentabilité adapté à la Suisse romande.

**Inputs :**

### 5.5 Goban Coach — Victor

**Route** : `/goban-coach`

Le Goban Coach est un outil unique : il te permet de jouer au jeu de Go et d'apprendre simultanément à penser comme un entrepreneur. Victor, l'agent IA, commente chaque coup en direct.

**Démarrage :**
1. Choisis ton niveau : **Débutant** / Intermédiaire / **Expert**
2. Choisis la taille du plateau : **9×9** (rapide) / 13×13 / 19×19
3. Active ou non le **Mode Carrière** : chaque coup est mis en parallèle avec une décision entrepreneuriale

**Comment jouer :**
- Clique sur une intersection du plateau pour poser une pierre (Noir)
- L'IA (Blanc) joue automatiquement
- **Victor commente en temps réel** (streaming token par token) : analyse de ton coup, intention derrière le coup adverse, conseil stratégique

**Commentaire adapté au niveau :**
- **Débutant** : 2-3 phrases simples, pas de termes japonais, conseil immédiat
- **Intermédiaire** : format complet (coup reçu / question d'intention / conseil / réflexion)
- **Expert** : + estimation de l'impact territorial (ex : "+2 pts Noir")

**Bouton "💡 Indice"** : demande à Victor de suggérer une zone à explorer avant de jouer ton coup.

**Audio** : sons d'ambiance et commentaires audio via Bunny CDN (bienvenue, fin de partie, règles).

**Métaphore carrière** (Mode Carrière activé) : chaque concept du Go est traduit en langage entrepreneurial — la capture d'un groupe = perdre un client majeur, le territoire = ta part de marché, etc.

### 6.6 Test d'audience — MirrorFisch

**Route** : `/mirrorfisch`

Simule la réaction de tes personas cibles face à un message marketing avant de le publier.

**Comment ça marche :**
1. Saisis ton message (post, email, accroche)
2. Sélectionne les personas à tester (profils prédéfinis selon ton marché)
3. L'IA simule la réaction de chaque persona :
   - Score de conversion (probabilité d'action)
   - Score d'engagement (intérêt, rétention)
   - Retours qualitatifs (ce qui accroche, ce qui rebute)

Utile avant de lancer une campagne pour ajuster le message.

## 8. Les 7 agents IA

Nova Solo dispose de 7 agents spécialisés, chacun avec un rôle distinct et un contexte suisse romand intégré.

### Nova — Coordinatrice générale

**Rôle** : Orientation, coordination, questions générales sur l'utilisation de l'app.  
**Idéal pour** : "Par où commencer ?", "Quelle est ma prochaine priorité ?", aide à la navigation.

---

### Strategist — Conseiller en stratégie

**Rôle** : Analyse de marché, positionnement, modèle économique, compétitivité.  
**Idéal pour** : Travailler le BMC, le Business Plan, définir sa niche, challenger ses hypothèses.  
**Utilisé par** : BMC, Business Plan, Diagnostic, Goban Coach (Victor).

---

### Financier — Expert financier

**Rôle** : Trésorerie, rentabilité, fiscalité suisse (AVS, TVA 8.1%, RI vs. SARL), levée de fonds.  
**Idéal pour** : Valider un business plan financier, optimiser sa structure de charges, préparer un entretien bancaire.  
**Utilisé par** : Finances, Pricing.

---

### Communicant — Expert en communication

**Rôle** : Personal branding, LinkedIn, copywriting, rédaction de contenu, ton et voix de marque.  
**Idéal pour** : Rédiger des posts, préparer un dossier de présentation, affiner son message.  
**Utilisé par** : Marketing, Dossier, CV.

---

### Commercial — Expert en vente

**Rôle** : Prospection, closing, gestion des objections, partenariats, SONCAS.  
**Idéal pour** : Préparer un entretien commercial, rédiger des emails de prospection, analyser un prospect.  
**Utilisé par** : Pipeline.

---

### Jurist — Conseiller juridique

**Rôle** : Forme juridique (RI vs. SARL), Registre du Commerce, nLPD (protection des données), contrats.  
**Idéal pour** : Choisir entre RI et SARL, comprendre ses obligations légales, rédiger des CGV.  
**Utilisé par** : Contrat.

> ⚠️ Les conseils du Jurist sont informatifs et ne remplacent pas un avocat.

---

### Technicien — Expert en outils & automatisation

**Rôle** : Stack technique, outils SaaS, CRM, site web, automatisation, IA.  
**Idéal pour** : Choisir ses outils, configurer des automatisations, intégrer une solution technique.

## 10. Réglages & personnalisation

## 11. Astuces & flux de travail recommandés

### Flux débutant (première semaine)

```
Réglages → Diagnostic → BMC → Business Plan → Pricing → CV
```

Le Diagnostic pré-remplit les 4 derniers modules. Tu affines ensuite chaque bloc à ton rythme.

### Flux hebdomadaire recommandé

```
Lundi      → Tableau de bord (rituels + KPIs)
Mardi      → Pipeline (relances, nouvelles fiches)
Mercredi   → Marketing (1 contenu généré et planifié)
Jeudi      → Finances (saisie des transactions de la semaine)
Vendredi   → Cabinet Hermès (question stratégique de la semaine)
```

### Préparer un entretien bancaire

```
1. Business Plan → toutes sections complètes
2. Finances → projections 12 mois exportées en CSV
3. CV → version Bancaire exportée en PDF
4. Financier (Hermès) → "Prépare-moi aux questions difficiles d'un chargé de crédit BCVS"
```

### Lancer une campagne marketing

```
1. MirrorFisch → tester le message avant publication
2. Marketing → générer les variantes de contenu
3. Pipeline → identifier les prospects à cibler en parallèle
4. Communicant (Hermès) → affiner le ton
```

### Gérer la dimension LACI

```
Tableau de bord → Jalons LACI (suivi des 6 étapes)
Jurist (Hermès) → questions sur l'art. 71a–71d
Business Plan → section "Forme juridique" + "Risques"
```

## Raccourcis & astuces générales

- **Sauvegarde automatique** : les formulaires sont sauvegardés en temps réel — pas besoin de cliquer sur "Enregistrer"
- **Persistance locale** : certains champs (CV, Hermès, calculateurs) utilisent le stockage local du navigateur — ils restent entre sessions même hors connexion
- **Exports** : tous les exports (PDF, Word, CSV, HTML) se téléchargent directement dans ton dossier Téléchargements
- **Raccourci chat** : cliquer sur le logo d'un agent dans le Cabinet Hermès ouvre directement une conversation avec cet agent
- **Mode nuit** : l'interface utilise un thème sombre par défaut pour les longues sessions de travail
